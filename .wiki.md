# Project Summary
The HSE Management Platform is a comprehensive safety reporting and management system tailored for the oil and gas industry. It enables real-time monitoring, reporting, and analysis of health, safety, and environmental (HSE) incidents, leveraging AI capabilities for improved decision-making and risk management.

# Project Module Description
- **Backend**: FastAPI application managing data models, routing, and AI integrations.
- **Frontend**: React-based user interface for reporting hazards and incidents, tracking actions, and displaying dashboards.

# Directory Tree
```
app/
├── backend/
│   ├── core/               # Core functionalities including authentication and database management
│   ├── data_models/        # JSON files defining data structures for actions, attachments, etc.
│   ├── dependencies/       # Dependency management for authentication and database
│   ├── middlewares/        # Middleware functionalities
│   ├── models/             # ORM models for database interactions
│   ├── requirements.txt     # Python dependencies
│   ├── routers/            # API route definitions
│   ├── services/           # Business logic and service layer
│   ├── main.py             # Entry point for the backend service
│   └── lambda_handler.py    # AWS Lambda handler for serverless deployment
└── frontend/
    ├── src/                # Source code for the frontend application
    ├── public/             # Public assets including favicon and robots.txt
    ├── package.json         # Node.js dependencies and scripts
    └── index.html          # Main HTML file for the frontend application
```

# File Description Inventory
- **backend/**: Contains the FastAPI application structure including models, routers, services, and configurations.
- **frontend/**: Contains the React application code, including components, pages, and styles.

# Technology Stack
- **Backend**: FastAPI, SQLAlchemy, Python
- **Frontend**: React, TypeScript, Tailwind CSS, Shadcn-UI
- **AI Integration**: Gemini-2.5-pro for text analysis
- **State Management**: React Query
- **Routing**: React Router

# Usage
1. **Install Dependencies**: 
   - For backend: `pip install -r requirements.txt`
   - For frontend: `pnpm install`
   
2. **Build the Project**:
   - Backend: Run the FastAPI application.
   - Frontend: Execute `pnpm run build` to compile the application.

3. **Run the Application**: Start the backend and frontend servers as per your deployment strategy.
